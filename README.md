## 搬瓦工迁移机房操作步骤

进 KiwiVM 控制面板，左侧菜单找到 **Migrate to another DC**，点进去。

1. 页面上会显示你当前所在的机房（Current location of this VPS）
2. 从列表里选一个目标机房
3. 点 **Confirm on next step**
4. 确认无误后点 **Start Migration** 开始迁移
5. 等待完成，一般几分钟到十几分钟，完成后会收到邮件通知

迁移期间 VPS 还是可以操作的，只是在最后阶段会短暂关机几秒到几分钟。完成后 IP 地址换了，其他都跟以前一样。

说句题外话：每次迁移会消耗一定流量，消耗量跟你 VPS 里的文件量有关，不是固定的。频繁来回切机房的话注意别把当月流量耗完了。

免费，不限次数。只要你的套餐支持迁移这个功能，想换几次换几次。

所以想通过换机房来绕过 IP 被封的问题，这条路走不通。被封的话，只能付费换 IP（大概 $8.79 一次），或者等两三个月看能不能自动解封。

## 常见问题（FAQ）

**Q：搬瓦工迁移机房会丢数据吗？**
A：不会。迁移是把整个 VPS 搬到新机房，系统、文件、配置全部保留，只有 IP 地址会变成新机房的。

**Q：搬瓦工迁移机房需要多长时间？**
A：一般几分钟到十几分钟，视 VPS 里的文件大小而定。迁移全程 VPS 基本可用，最后关机阶段几秒到几分钟。

**Q：我的 IP 被封了，可以靠迁移换个新 IP 吗？**
A：不行。搬瓦工明确规定 IP 被封之后迁移功能自动禁用。只能付费换 IP（约 $8.79），或者等待解封。

**Q：KVM 套餐能迁到 CN2 GIA 机房吗？**
A：不能。每个套餐有固定的可迁移机房范围，KVM 套餐只能在普通线路机房之间切换，CN2 GIA 机房只有 CN2 GIA 系列套餐才能用。

**Q：迁移机房会消耗流量吗？**
A：会，每次迁移会消耗一部分流量，消耗量跟 VPS 里的数据量有关。频繁迁移注意别把月流量用完。
